edX REST Framework Extensions
=============================

This package provides extensions for `Django REST Framework <http://www.django-rest-framework.org/>`_ that are useful
for developing on the edX platform.

Requirements
------------

* Python (2.7, 3.5, 3.6)
* Django (1.8, 1.9, 1.10, 1.11)
* Django REST Framework (3.2+)

Installation
------------

Install using ``pip``:

.. code-block:: bash

    $ pip install edx-drf-extensions
